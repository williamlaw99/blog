```

```

void	无返回值
```



#### 访问控制相关-4个

```
private	私有的
protected	受保护的
public	公共的
default	默认
```



#### 错误处理类-6个

```
assert	断言表达式是否为真
catch	捕捉异常
finally	有没有异常都执行
throw	抛出一个异常对象
throws	声明一个异常可能被抛出
try	捕获异常
```

```
